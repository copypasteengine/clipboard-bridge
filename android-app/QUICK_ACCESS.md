# 🚀 Android 快捷访问指南

像 iOS 快捷指令一样，Android 也可以实现**一键同步**，无需打开 App！

## 🎯 三种快捷访问方式

### 方式 1: 桌面小部件（最推荐 ⭐⭐⭐）

**一键操作，就在主屏幕上！**

#### 添加步骤

1. **长按主屏幕空白处**
2. 点击**"小部件"**（或"Widgets"）
3. 找到**"Clipboard Bridge"**
4. 拖动小部件到主屏幕
5. ✓ 完成！

#### 小部件界面

```
┌──────────────────┐
│ Clipboard Bridge │
├──────────────────┤
│  [ ⬇️ ]  [ ⬆️ ]  │
│  [ 🔄 智能同步 ] │
└──────────────────┘
```

- **⬇️** - 从电脑获取
- **⬆️** - 发送到电脑  
- **🔄** - 智能同步

#### 使用方法

**场景：复制内容后立即发送到电脑**
```
1. 在任意 App 中复制内容
2. 回到主屏幕
3. 点击小部件的 ⬆️ 按钮
4. ✓ 已发送！（显示 Toast 提示）
```

**完全不需要打开 App！**

---

### 方式 2: App Shortcuts（长按图标）

**长按应用图标，快速选择操作！**

#### 使用步骤

1. **长按** App 图标
2. 弹出快捷菜单：
   ```
   🔄 智能同步
   ⬇️ 从电脑获取
   ⬆️ 发送到电脑
   ```
3. 点击对应操作
4. ✓ 立即执行！

#### 创建主屏幕快捷方式

1. 长按 App 图标
2. 长按菜单中的某个操作（如"发送到电脑"）
3. 拖动到主屏幕
4. ✓ 创建独立快捷图标！

现在主屏幕上有**独立的"发送到电脑"图标**，点击即可！

---

### 方式 3: 快速设置磁贴（最快 ⭐⭐⭐⭐⭐）

**下拉通知栏，一键智能同步！**

#### 添加步骤

1. **下拉通知栏**两次（完全展开）
2. 点击**铅笔图标**（编辑按钮）
3. 找到**"Smart Sync"**磁贴
4. 拖动到快速设置区域
5. 点击**"完成"**

#### 使用方法

**最快的同步方式：**
```
1. 复制内容
2. 下拉通知栏
3. 点击"Smart Sync"磁贴
4. ✓ 自动同步！
```

**比 iOS 快捷指令还快！**

## 🔧 常见问题

### Q1: 小部件点击没反应

**A:** 检查：
- App 是否已配置服务器地址
- 在 App 中测试同步是否正常
- 重新添加小部件

### Q2: 快速设置磁贴在哪里？

**A:** 
- 完全下拉通知栏（下拉两次）
- 点击铅笔图标编辑
- 向下滚动找到"Smart Sync"
- 拖动到上方

### Q3: App Shortcuts 没有显示

**A:**
- 确保是 Android 7.1 以上
- 重新安装 App
- 长按图标稍等片刻

### Q4: 同步失败

**A:**
- 检查服务器配置（在 App 中打开设置）
- 确保手机和电脑在同一网络
- 查看 Toast 提示的错误信息
