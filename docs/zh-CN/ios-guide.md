# 📱 iOS 快捷指令指南

使用 iOS 快捷指令配合 Clipboard Bridge 的完整指南。

> **提示：** 如果你使用 iPhone + Mac，请使用 Apple 通用剪贴板。

## 📥 基础配置

### 从电脑获取

1. 打开"快捷指令" App → 点击"+"
2. 添加动作："获取 URL 的内容"
   - URL：`http://192.168.1.100:5678/pull`（替换为你的电脑 IP）
   - 方法：GET
   - 如果设置了 Token：添加标头 `X-Auth-Token` = `your-token`
3. 添加动作："设定剪贴板"
4. 添加动作："显示通知" → "✓ 已从电脑同步"
5. 保存快捷指令

### 发送到电脑

1. 打开"快捷指令" App → 点击"+"
2. 添加动作："获取剪贴板"
3. 添加动作："获取 URL 的内容"
   - URL：`http://192.168.1.100:5678/push`
   - 方法：POST
   - 请求体：表单
   - 添加字段：`text` = [剪贴板]
   - 如果设置了 Token：添加标头 `X-Auth-Token` = `your-token`
4. 添加动作："显示通知" → "✓ 已发送到电脑"
5. 保存快捷指令

## 🎯 智能同步（推荐）

自动检测同步方向：

1. 通过 `/meta` 获取电脑剪贴板
2. 获取 iPhone 剪贴板
3. 比较并决定：
   - 相同 → 显示"已同步"
   - iPhone 为空 → 从电脑获取
   - 电脑为空 → 发送到电脑
   - 不同 → 显示菜单选择

详细步骤见其他文档。

## ⚡ 快速访问

### 主屏幕小组件

1. 长按主屏幕
2. 点击"+"
3. 搜索"快捷指令"
4. 添加小组件
5. 选择你的剪贴板同步快捷指令

### Siri 语音控制

对 Siri 说："嘿 Siri，运行剪贴板同步"

### 自动化

在快捷指令 App 创建自动化：

**触发器：** 连接到 WiFi
- 选择家庭/办公室 WiFi
- 操作：运行剪贴板同步快捷指令
- 关闭"运行前询问"

**触发器：** 打开特定 App
- 例如：备忘录、Safari
- 操作：从电脑获取
- 打开 App 时自动运行
