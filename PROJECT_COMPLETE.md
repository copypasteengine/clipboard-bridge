## 🎯 项目定位

### ✅ 专注于文本同步

**支持的内容：**
- 纯文本、URL、验证码、消息、代码片段

**不支持的内容：**
- 图片、文件、富文本、二进制数据

**推荐替代方案：**
- 图片/文件传输 → [LocalSend](https://localsend.org/)
- 功能全面的同步 → [KDE Connect](https://kdeconnect.kde.org/)

### 🎯 目标用户

**主要用户：**
- 需要快速传输文字的用户
- 开发者（代码片段、命令）
- 文字工作者
- 日常使用（验证码、链接）

**不适合：**
- 需要传输图片的用户
- 需要传输文件的用户
- 需要富文本格式的用户

## 🚀 用户体验对比

### Android 用户（最佳体验 ⭐⭐⭐⭐⭐）

**自动同步（Android → 电脑）：**
```
复制文本 → (自动发送) → 电脑粘贴
```
**类似度：90%**（接近 Apple Universal Clipboard）

**手动同步（电脑 → Android）：**
```
电脑复制 → 下拉点击 → 手机粘贴
```
**只需 1 步！**

### iOS 用户（实用体验 ⭐⭐⭐⭐）

**快捷指令方式：**
```
复制 → 回主屏幕 → 点击 → 完成
```
**2-3 步**

**Siri 语音：**
```
复制 → "嘿 Siri，同步剪贴板" → 完成
```
**无需点击屏幕**
