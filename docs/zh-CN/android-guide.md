# 📱 Android 使用指南

Android 端完整使用手册。

## 📥 安装

1. 从 [Releases](https://github.com/copypasteengine/clipboard-bridge/releases) 下载 `clipboard-bridge-android-*.apk`
2. 在设置中允许"安装未知应用"
3. 安装 APK
4. 打开应用

## ⚙️ 初次配置

1. 点击右上角 ⚙️（设置）图标
2. 输入电脑地址：`http://192.168.1.100:5678`
3. （可选）如果配置了 Token，输入 Token
4. 点击"保存"
5. 连接状态显示"已连接" ✓

## 🎯 基本使用

### 手动同步

**从电脑获取：**
1. 点击"从电脑获取"按钮
2. 电脑剪贴板内容复制到手机
3. 在任意位置粘贴 ✓

**发送到电脑：**
1. 在手机上复制文本
2. 点击"发送到电脑"按钮
3. 内容已发送到电脑 ✓

**智能同步：**
1. 点击"智能同步"按钮
2. App 自动检测同步方向
3. 完成 ✓

### 自动同步（可选）

启用自动同步，复制时自动发送到电脑：

1. 在 App 中，打开"自动同步"开关
2. 授予通知权限
3. 现在复制内容会自动同步到电脑！

**注意：** 会显示"自动同步已开启"通知

## ⚡ 快捷访问（推荐！）

### 快速设置磁贴（最快）

**设置：**
1. 完全下拉通知栏（下拉两次）
2. 点击铅笔图标（编辑）
3. 找到"Smart Sync"磁贴
4. 拖到快速设置区域
5. 完成！

**使用：**
```
复制 → 下拉 → 点击"Smart Sync" → 完成！
```

### 桌面小部件

**设置：**
1. 长按主屏幕
2. 点击"小部件"
3. 找到"Clipboard Bridge"
4. 拖到主屏幕

**小部件有 3 个按钮：**
- ⬇️ 从电脑获取
- ⬆️ 发送到电脑
- 🔄 智能同步

### App Shortcuts

**使用：**
1. 长按应用图标
2. 从菜单选择操作
3. 完成！

**创建独立快捷方式：**
- 长按图标 → 长按菜单项 → 拖到主屏幕

详见 [快捷访问指南](./QUICK_ACCESS.md)

## 🐛 故障排查

**无法连接：**
- 确保手机和电脑在同一 WiFi
- 检查电脑防火墙允许 5678 端口
- 验证 IP 地址正确
- 确认桌面服务正在运行

**Token 错误：**
- 确保 Token 与 config.json 中一致
- Token 区分大小写

**自动同步不工作：**
- 检查通知权限已授予
- 关闭应用的电池优化
- 确保服务运行中（通知可见）

更多问题见 [常见问题](./faq.md)
