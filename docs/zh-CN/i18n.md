# 🌐 Clipboard Bridge - 国际化支持

## 📱 Android App

### 支持的语言

- 🇺🇸 **English** (默认)
- 🇨🇳 **简体中文**
- 🇯🇵 **日本語**

### 语言自动检测

Android App 会自动根据系统语言显示对应界面：

| 系统语言 | 显示语言 |
|----------|----------|
| Chinese (China) | 简体中文 |
| Chinese (Taiwan/Hong Kong) | English |
| Japanese | 日本語 |
| 其他 | English |

### 切换语言

在 Android 系统设置中更改语言：

```
设置 → 系统 → 语言和输入法 → 语言
```

App 会自动跟随系统语言。

### 添加新语言

如果你想贡献其他语言翻译：

1. 复制 `app/src/main/res/values/strings.xml`
2. 创建新的语言目录：
   ```
   values-fr/strings.xml      # 法语
   values-de/strings.xml      # 德语
   values-es/strings.xml      # 西班牙语
   values-ko/strings.xml      # 韩语
   values-ru/strings.xml      # 俄语
   ```
3. 翻译所有字符串
4. 提交 Pull Request

**语言代码参考：**
- `values` - English (默认)
- `values-zh-rCN` - 简体中文
- `values-zh-rTW` - 繁体中文
- `values-ja` - 日语
- `values-ko` - 韩语
- `values-fr` - 法语
- `values-de` - 德语
- `values-es` - 西班牙语
- `values-ru` - 俄语

## 🖥️ 桌面服务

### 支持的语言

- 🇺🇸 **English** (默认)
- 🇨🇳 **简体中文**
- 🇯🇵 **日本語**

### 语言检测

桌面服务会按以下顺序检测语言：

1. 环境变量 `LANG`
2. 环境变量 `LANGUAGE`
3. 环境变量 `LC_ALL`
4. 环境变量 `LC_MESSAGES`
5. 默认使用 English

**检测规则：**
- 包含 `zh` → 简体中文
- 包含 `ja` → 日本語
- 其他 → English

### 手动设置语言

**Linux/macOS:**
```bash
export LANG=zh_CN.UTF-8  # 中文
export LANG=ja_JP.UTF-8  # 日语
export LANG=en_US.UTF-8  # 英文

./clipboard-bridge
```

**Windows (PowerShell):**
```powershell
$env:LANG="zh_CN.UTF-8"  # 中文
.\clipboard-bridge.exe
```

### 添加新语言

贡献新语言翻译：

1. 编辑 `i18n.go`
2. 在 `translations` 中添加新语言：
   ```go
   LangFR = "fr" // 法语
   
   translations[LangFR] = map[string]string{
       "app_title": "Pont de presse-papiers",
       // ... 其他翻译
   }
   ```
3. 更新 `detectSystemLanguage()` 函数
4. 提交 Pull Request

## 📝 翻译指南

### 翻译原则

1. **保持一致性** - 相同功能使用相同术语
2. **符合习惯** - 使用目标语言的自然表达
3. **简洁明了** - 界面文字尽量简短
4. **保留表情符号** - Emoji 是通用的，不需翻译
